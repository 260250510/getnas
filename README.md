# 搞个 NAS

本项目将指导你从零开始构建一台家庭 NAS 存储服务器，从硬件到系统再到程序，完全自己动手搞定。

## 构建目标

遵照本项目的指导，你最终会将一台普通的计算机（PC）打造成为具有以下功能的家庭存储服务器：

* 多用户私有网盘
* HTTP、BT 全能下载机
* 局域网文件共享
* 跨平台文件同步
* Apple TimeMachine

搭配云服务器、国际域名、邮件推送等云计算资源，将能实现以下功能：

* 通过互联网访问网盘等功能
* 远程管理服务器
* 离线下载
* 基于 P2P 的虚拟专用网
* E-mail 故障通知

## 适用群体

原则上本方案适用于任何人，我们会尽可能提供详细的操作说明，没有 Linux 系统使用经验的用户可能需要额外掌握一些知识，因此构建过程会相应延长。
